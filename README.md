# 🚀 FoundersFuel: Premium AI Startup Mentor

FoundersFuel is a highly polished, autonomous startup advisory platform designed for founders who need domain-specific expertise without the "generic assistant" noise.

![Landing Preview](https://via.placeholder.com/1200x600/0a0c10/7c3aed?text=FoundersFuel+Premium+AI+Interface)

## 🎯 The Niche: AI Startup Mentor
Generic AI often gives safe, surface-level advice. **FoundersFuel** is built to think like a battle-hardened VC and a technical co-founder. It doesn't just answer questions; it challenges assumptions, vettes unit economics, and builds roadmaps.

### 🎭 Persona Modes
- **Standard Mentor**: Balanced, encouraging yet strategic.
- **Brutal Scrubber**: A high-friction reviewer designed to find every flaw in a pitch.
- **GTM Architect**: Focused purely on distribution, cycles, and growth loops.

## 🛠️ Technology Stack
- **Frontend Architecture**: React 19 + Vite for ultra-fast performance.
- **Styling Engine**: Tailwind CSS v4 (CSS-first approach) for precision layouts.
- **Motion & UX**: Framer Motion for immersive transitions and micro-interactions.
- **Content Delivery**: React-Markdown for structured, readable legal/strategic output.
- **Icons**: Lucide React for consistent, modern visual language.

## ✨ Premium Frontend Details
- **Glassmorphism UI**: Deep, blurred panels that create a sense of depth and hierarchy.
- **Context-Aware Input**: Intelligent multi-line input with keyboard shortcuts and focus effects.
- **Seamless State Transitions**: Zero-flicker routing between landing and chat.
- **Dynamic Personas**: Real-time context switching that changes the "soul" of the AI.
- **Responsive Mastery**: Fully optimized for everything from mobile brainstorming to desktop pitch-deck scrubbing.

## 🧠 System Prompt Philosophy
Rather than simple "You are an assistant," FoundersFuel uses a structured knowledge matrix:
1. **Role**: Senior Venture Partner & Multi-exit Founder.
2. **Constraints**: No fluff, prioritize data over opinions, use bulleted action items.
3. **Knowledge Base**: Deep understanding of Zero to One (Thiel), Lean Startup (Ries), and modern SaaS metrics.

## 🚀 Getting Started
1. Clone the repo
2. `npm install`
3. `npm run dev`
